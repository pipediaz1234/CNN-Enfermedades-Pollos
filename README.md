Implementación de una CNN desde cero con TensorFlow/Keras para clasificar automáticamente 
enfermedades avícolas a partir de imágenes. Dataset de 8.067 imágenes distribuidas en 
4 clases: Healthy, Coccidiosis, Salmonella y New Castle Disease.

Arquitectura de 4 bloques Conv2D con BatchNormalization, MaxPooling y Dropout.
Técnicas aplicadas: Data Augmentation, EarlyStopping y ReduceLROnPlateau.
Entorno: Google Colab GPU T4.
